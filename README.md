# HRMS Microservices

A Human Resource Management System built using Spring Boot Microservices.

## Modules

- Auth Service
- Employee Service
- Department Service
- Leave Service
- Attendance Service
- Payroll Service
- API Gateway
- Config Server
- Eureka Server

## Technologies

- Java 24
- Spring Boot
- Spring Security
- JWT Authentication
- Spring Data JPA
- PostgreSQL
- Maven
- Eureka Server
- Spring Cloud Gateway
- Config Server

## Features

- User Authentication
- Role Based Authorization
- Employee Management
- Attendance Management
- Leave Management
- Payroll Management
